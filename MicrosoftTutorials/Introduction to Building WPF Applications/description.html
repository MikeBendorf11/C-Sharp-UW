This sample provides an introduction to the development of a Windows Presentation Foundation (WPF) application that includes the elements that are common to most WPF applications: Extensible Application Markup Language (XAML) markup, code-behind, application definitions, controls, layout, data binding, and styles.

The walkthrough for this sample(http://msdn.microsoft.com/en-us/library/ms752299(VS.100).aspx) guides you through the development of a simple WPF application using the following steps.

  • Defining XAML to design the appearance of the application's user interface (UI).

  • Writing code to build the application's behavior.

  • Creating an application definition to manage the application.

  • Adding controls and creating the layout to compose the application UI.

  • Creating styles to create a consistent appearance throughout an application's UI.

  • Binding the UI to data to both populate the UI from data and keep the data and UI synchronized.
